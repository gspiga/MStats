1. Generate data sets of \(n\) observations from normally distributed random variables using \(n = 10, 100, 1000, 10000\).

Generate data sets of \(n\) observations from normally distributed random variables using \(n = 10, 100, 1000, 10000\).

# By setting a seed, we are able to make sure Rstudio keeps our results consistent when we randomly generate numbers
set.seed(123)

# We can take advantage of the rnorm function here
# Default is mean =0, sd = 1, but you can always edit this to your needs
n10 = rnorm(10, mean = 0, sd = 1) 
n100 <- rnorm(100)
n1000 <- rnorm(1000)
n10000 <- rnorm(10000)

{
qqnorm(student$height, main = "(a) Normal Probability Plot for Student's Height")
qqline(student$height)
}

hist(student$height, col = "orange")

Most of the points match up along a straight line. It suggests students heights are approximately normally distributed.

# (b)
{
qqnorm(student$pulse, main = "(b) Normal Probability Plot for Student's Pulse")
qqline(student$pulse)
}

hist(student$pulse, col = "green")

The points forms a (convex) curve instead of a straight line. It indicates a right-skewed distribution. It does not suggest students pulses are approximately not normally distributed.

Bonus - Changing your RStudio Theme

You can customize your RStudio color layout with built-in RStudio themes! Here is how to do it.

  • On the top left, click on Tools

  • Scroll down to Global Options

  • On the left, click on the Appearence option

  • Scroll through the list of Editor Themes and click Apply once you have chosen one. Once you do, your RStudio will have a new look! If you would like to get into the spirit of Halloween, I recommend the Merbivore theme.
